==============================================================================================
 NOTICE D'UTILISATION DES CARTES D'ORACLES
==============================================================================================


Ce deck de cartes est un outil de génération d'idées et d'inspiration narrative, parfait pour les Maîtres de Jeu (MJ) ou pour jouer en solo/coopératif sans MJ. Chaque carte est conçue pour poser une question ou enrichir une scène, un lieu, ou un personnage non-joueur (PNJ).

---

[ 1 ] LA STRUCTURE DE LA CARTE
==============================

Chaque carte générée est un concentré d'éléments narratifs.

+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Champ          | Description                                                   | Usage Narratif Principal                                                      |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Numéro & Titre | Le numéro de la carte et la réponse centrale (ex: Oui, mais…, | Réponse à une Question : Le cœur de l'oracle.                                 |
|               | Non, et…).                                                    |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Symbole        | Un emblème figuratif (ex: Dragon, Clef, Torche).              | Thème & Focalisation : Indique l'objet ou le concept central de la scène.     |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Verbes         | 3 actions/concepts (ex: Vole, Confronte, Raconte).            | Action : Les forces actives dans la scène, ce qui se passe ou doit se passer. |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Lieu           | Un environnement (ex: Auberge de bord de route,               | Contexte : Où l'action se déroule-t-elle ?                                    |
|               | Grotte humide).                                               |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Personnage     | Un archétype humain ou un monstre (ex: Marchand / Squelette,  | Acteur : Qui est impliqué ?                                                   |
|               | Sorcier / Géant).                                             |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Objet          | Un item concret (ex: Amulette, Tissu ensanglanté,             | Item : L'élément matériel que l'on cherche,                                   |
|               | Outil de voleur).                                             | trouve ou qui est important.                                                  |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Motivation     | Une force intérieure ou un but (ex: Vengeance, Ambition,      | But du PNJ : Pourquoi le PNJ agit-il ?                                        |
|               | Rédemption).                                                  |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Traits         | 3 adjectifs descriptifs (ex: Coloré, Toxique,                 | Ambiance & Description : Qualifie l'objet, le lieu ou l'ambiance générale.    |
|               | Brinquebalant).                                               |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Secret         | Un lourd passé (ex: A tué un innocent, A passé un pacte       | Sombre Secret : Potentiel d'intrigue autour d'un PNJ.                         |
|               | avec une entité maléfique).                                   |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Relation       | Le lien entre un PJ et un PNJ (ex: Rival, Mentor,             | Lien Direct : Comment le PNJ est-il connecté au groupe ou à un joueur ?       |
|               | Créancier).                                                   |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+
|Réaction       | Une paire amical/hostile (ex: Curieux / Méfiant,              | Comportement du PNJ : Comment le PNJ se comporte-t-il initialement ?          |
|               | Généreux / Égoïste).                                          |                                                                               |
+---------------+---------------------------------------------------------------+-------------------------------------------------------------------------------+

[ 2 ] MODES D'UTILISATION RECOMMANDÉS
=====================================


>> [A] L'ORACLE (RÉPONDRE À UNE QUESTION)
-----------------------------------------

Tirez une carte pour répondre à une question fermée (Oui/Non) posée par les joueurs ou le MJ.

 1) Posez une question nécessitant une réponse simple
    (ex: "Le garde nous laisse-t-il passer ?").
 2) Tirez une carte.
 3) Lisez le Titre pour la réponse:

+-----------------------------+--------------------------------------------------------------------------+
| FORME                       | DESCRIPTION                                                              |
+-----------------------------+--------------------------------------------------------------------------+
| Oui / Non                   | Réponse directe.                                                         |
+-----------------------------+--------------------------------------------------------------------------+
| Oui, mais… / Non, mais…     | La réponse est celle indiquée, mais la carte révèle une COMPLICATION     |
|                             | (le "mais").                                                             |
+-----------------------------+--------------------------------------------------------------------------+
| Oui, et… / Non, et…         | La réponse est celle indiquée, et la carte ajoute une OPPORTUNITÉ /      |
|                             | CONSÉQUENCE (le "et").                                                   |
+-----------------------------+--------------------------------------------------------------------------+


>> [B] CRÉATION DE SCÈNES & D'INTRIGUES
---------------------------------------

Tirez une carte pour générer une rencontre ou pour créer une nouvelle scène lorsque l'intrigue piétine.

- Scène de Rencontre :
  -> Lieu       : décor de la scène.
  -> Personnage : acteur principal.
  -> Objet      : L'item que le PNJ veut, que les PJ trouvent, ou qui est au cœur du conflit.
  -> Verbes     : ce qui se passe ou doit se passer.
  -> Secret     : intrigue sous-jacente que les joueurs pourraient découvrir.

- Défaut de PNJ : Générer un PNJ mémorable à la volée.

    * Identité    : champ Personnage.
    * Intentions  : champ Motivation.
    * Interaction : champ Relation (lien au groupe) et champ Réaction
                    (comportement initial).
    * Complexité  : Traits (personnalité / apparence) + Secret (talon d'Achille).


>> [C] AMBIANCE & DESCRIPTION
------------------------------

Utilisez les champs descriptifs pour enrichir un lieu ou un objet existant :

- Décrivez une Tour de mage (Lieu) comme étant :
    -> Tarabiscotée
	ou/et
    -> Brumeuse
	ou/et
    -> Maudite
  (Traits)

- Imaginez un PNJ Guerrier / Orque (Personnage) qui, en réalité :
    -> A trahi quelqu'un pour survivre (Secret)
    -> Cherche la Rédemption (Motivation)

Exemple de carte
----------------------------------------------------------------------------------------------
Carte 1 — Non
Symbole : Dague
Verbes : Embrasse, Creuse, Soulève
Lieu : Champs labourés
Personnage : Sorcier / Géant
Objet : Pièce d'or
Motivation : Connaissance
Traits : Noir, Volé, Facile
Secret : Est responsable d’un incendie meurtrier
Relation : Guide
Réaction (amical/hostile) : Récalcitrant / Violent
